# 💸 Velora Finance

**Velora Finance** is a modern personal finance web app built with **Vanilla JavaScript**, designed to help users track their income, expenses, budgets, and financial goals in a clean and interactive dashboard.

---

## 🚀 Features

* 📊 **Dashboard Overview**

  * Total Balance
  * Monthly Income & Expenses
  * Budget Left
  * Financial Health Score
  * Charts & Insights

* 💳 **Transactions**

  * Add Income & Expenses
  * View recent transactions
  * Track financial activity easily

* 💰 **Budgets**

  * Create custom budgets
  * Track spending progress
  * Visual indicators for budget usage

* 🎯 **Goals**

  * Set saving goals
  * Add funds toward goals
  * Track progress visually

* 🔐 **Authentication System**

  * Multiple account support
  * Switch between accounts

* 🌙 **Dark / Light Mode**

  * Smooth UI theme switching

---

## 🖥️ Best Viewing Experience

⚠️ This app is currently optimized for **Desktop screens only**.
Mobile responsiveness is under improvement.

---

## 📦 How to Use

### 1. Dashboard

* View your total balance, income, and expenses
* Analyze your financial health
* Check recent transactions and insights

---

### 2. Transactions

👉 Add a transaction:

* Click **“Add Expense”** or **“Add Income”**
* Enter:

  * Description
  * Amount
* Click **Save**

✔️ Your balance updates automatically

---

### 3. Budgets

👉 Create a budget:

* Go to **Budgets page**
* Click **“+ Add Budget”**
* Enter:

  * Budget Name
  * Budget Amount

✔️ Track:

* Spending progress
* Remaining budget
* Visual progress bars

---

### 4. Goals

👉 Create a goal:

* Go to **Goals page**
* Add a new goal (e.g., Laptop, Savings)

👉 Manage goal:

* Add funds
* Track completion progress
* Delete goal if needed

---

## 🛠️ Tech Stack

* HTML5
* CSS3 (Glassmorphism UI + Dark Mode)
* Vanilla JavaScript
* Chart.js (for analytics & graphs)
